Teste de Validação de QR Code por Empresa

  1. Abra o dashboard e vá para a aba "Compras"
  2. Clique em "Iniciar Câmera" para ativar a leitura
  3. Apresente um dos QR Codes abaixo para a câmera
  4. O sistema validará automaticamente se a empresa existe
  5. Confirme a compra apenas se a validação for bem-sucedida

Como Funciona a Validação:

  • QR Code Válido: Contém nome de empresa cadastrada no sistema
  • QR Code Inválido: Empresa não encontrada - mostra erro
  • Formato Suportado: Texto simples ou JSON
  • Validação: Case-insensitive (não diferencia maiúsculas/minúsculas)

Como Funciona a Validação

1. Iniciar Câmera

Clique no botão "Iniciar Câmera" para ativar a leitura

2. Apresentar QR Code

Posicione um dos QR Codes na área da câmera

3. Validação Automática

O sistema verifica se a empresa existe no banco de dados

4. Confirmação

Se válido, confirme a compra. Se inválido, erro será exibido

Cenários de Teste

QR Code Válido

QR Codes com nomes de empresas cadastradas serão aceitos

QR Code Inválido
